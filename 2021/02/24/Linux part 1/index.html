3.Linux目录结构

​ [ 树形结构]

  • /：是所有文件的根目录。

  • /bin：Binary的缩写，这个目录存放着经常使用的命令。

  • /home：Linux下的所有普通用户的子目录。

  • /lib：Library的缩写，存放系统最基本的动态连接共享库，目录中文件相当于Windows下的ddl文件。

  • /etc：存放所有系统管理所需要的配置文件的子目录。

  • /dev：Device的缩写，存放linux的外部设备，访问方式与文件夹相同。

  • /boot：系统内核及启动有关的文件。

  • /usr：一个重要且庞大的目录，存放系统应用程序及文档、用户安装的程序，类似于Windows下的program files 。

  • /proc：虚拟目录，时系统内存的映射，可直接访问这个目录来获取系统信息。

  • /media：Linux会把识别的设备，如U盘、光驱、声卡等，挂载到这个目录下。

  • /root：该目录为系统管理员，也称作超级权限者的用户主目录。

  • /var：这个目录中存放不断扩充的东西，包括各种日志文件。

  • /tmp：存放各种临时文件，是所有用户均可访问的地点。

4.用户目录

  • 绝对路径：从根目录开始写 /home/fossette

    通过pwd命令查看当前路径：

  • 相对路径：从当前目录开始到指定位置的相对路径。

    • cd . -> 当前目录

    • cd .. -> 返回上一级目录

    • cd - -> 跳到之前所在的目录

    • cd ~ -> 跳到自己的home目录

  • 解释fossette@ubuntu:~$

    fossette：当前登录的用户

    @：at 在

    ubuntu：主机名

    ~：用户的家目录（宿主目录)

    $：代表当前用户为普通用户

    #：代表当前用户为超级用户（root用户）

    切换为root用户的命令：sudo su，切换回：exit

创建单个目录 ：mkdir dirName 

创建嵌套目录 ：mkdir -p dir1/dir2/dir3 

删除空目录 : rmdir -p dirName (rm：remove)

彻底删除 : rm -r dirName  (-r递归) 

彻底删除 ：rm -ri dirName  (-ri提醒)

在当前目录下创建新文件: touch dirName
删除当前目录下的文件：rm -i dirName

cat dirName
退出：q

head -5:查看文件的前5行
tail:默认查看文件的后10行
tail -5:查看文件的后5行

复制xxx文件到xxfile中,如果不存在则创建xxfile : cp xxx xxfile 

复制dir1目录到newdir中，如果不存在则创建newfile : cp dir1 newfile

移动xxx文件到xxfile目录中: mv xxx xxfile 

给file文件创建硬链接: ln file file1
创建软链接的方式: ln -s [带绝对路径的文件]

      硬链接相当于一个备份，file1与file指向同一个文件，硬链接不占用磁盘空间。

      软链接约等于快捷方式（需要指定文件路径），硬链接约等于给文件起别名。
